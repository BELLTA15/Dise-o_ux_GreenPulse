# Paleta oficial GreenPulse

Usar esta paleta como base fija del proyecto (MVP y siguientes iteraciones).

## Colores base

- Primario: `#2D6A4F`
- Secundario: `#40916C`
- Acento: `#52B788`
- Fondo general: `#F6FAF8`
- Superficie (cards, appbar): `#FFFFFF`
- Texto principal: `#1B4332`
- Texto secundario: `#4F6D5E`
- Bordes suaves: `#D8E7DF`

## Uso recomendado

- `Primario`: botones principales, encabezados importantes, elementos activos.
- `Secundario`: estados seleccionados y componentes de apoyo.
- `Acento`: iconos destacados y llamadas visuales sutiles.
- `Fondo general` + `Superficie`: mantener apariencia limpia y profesional.
- `Texto principal` / `Texto secundario`: jerarquía de lectura clara.

## Nota

Esta paleta ya está integrada en la app en `lib/main.dart` mediante la clase `AppPalette`.
